**Theorem (Finite Verification).** For every natural number $n \leq 2540160$, the digit factorial sum of $n$ equals $n$ if and only if $n \in \{1, 2, 145, 40585\}$. That is,

$$\forall n \in \mathbb{N},\; n \leq 2540160 \implies \bigl(\sigma(n) = n \iff n = 1 \lor n = 2 \lor n = 145 \lor n = 40585\bigr),$$

where $\sigma(n)$ denotes the sum of the factorials of the decimal digits of $n$.
